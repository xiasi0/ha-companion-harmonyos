# HA Companion For HarmonyOS v1.1.0

This release focuses on sensor management, settings usability, theme behavior, and build compatibility.

## New Features

- Added broader HarmonyOS sensor updates and runtime state handling.
- Added sensor search and filter controls to make sensor settings easier to navigate.
- Added live settings state rows and switch rows for settings surfaces.
- Added support for the Do Not Disturb state permission used by related sensor/status features.
- Added Harmony Space Super Desktop allowlist support and non-P-gear usage capability.

## Improvements

- Refined settings button card styling.
- Improved settings title bar button sizing and layout.
- Improved dark theme background colors.
- Changed the default theme behavior to follow the system theme.
- Updated build profile SDK versions to HarmonyOS `6.1.0(23)`.
- Improved build script handling for SDK profile version normalization and clean rebuilds when profile SDK values change.

## Fixes

- Fixed empty home network summary handling.
- Fixed settings switch and live state presentation issues.
- Fixed build profile SDK version mismatch issues.

## Distribution Notes

- AGC builds are intended for AppGallery Connect signing/distribution workflows.
- GitHub builds are provided as unsigned artifacts for developers and users who want to sign with their own HarmonyOS certificate/profile.
- For learning how to install HarmonyOS artifacts, see: https://github.com/likuai2010/auto-installer
